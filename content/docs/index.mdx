---
title: Crumble
description: Hand-drawn React components for the web.
---

Crumble is a shadcn-style registry of wobbly, sketchy UI components built on [Rough.js](https://roughjs.com). Every component ships as copyable source. No black-box npm package, no lock-in.

```bash
npx shadcn add https://crumble.dev/r/button.json
```

The CLI copies the component source into your project and installs `roughjs` automatically.
