## Custom format

Use `formatValue` to control how the displayed value is rendered. Pass it from a client component to avoid the server/client serialization boundary.

<PreviewContainer
  componentName="progress"
  code={`<Progress
  label="Storage"
  value={3.2}
  max={5}
  showValue
  formatValue={(v) => \`\${v.toFixed(1)} GB\`}
/>`}
>
  <ProgressStorageDemo />
</PreviewContainer>
