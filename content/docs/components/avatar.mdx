## Stacked group

`AvatarGroup` takes an `avatars` array of props objects and a `max` count. Extra avatars show as a `+N` overflow badge.

<PreviewContainer
  componentName="avatar"
  code={`<AvatarGroup
  max={3}
  avatars={[
    { fallback: "AB" },
    { fallback: "CD" },
    { fallback: "EF" },
    { fallback: "GH" },
    { fallback: "IJ" },
  ]}
/>`}
>
  <AvatarGroup
    max={3}
    avatars={[
      { fallback: "AB" },
