## Installation

```bash
npx shadcn add https://crumble.dev/r/textarea.json
```

## Auto-grow

The textarea expands as you type. The SVG border tracks the height via `ResizeObserver`.

<PreviewContainer
  componentName="textarea"
  code={`<Textarea label="Notes" placeholder="Starts small, grows as you type..." autoGrow />`}
>
  <div className="w-full max-w-xs">
    <Textarea
      label="Notes"
      placeholder="Starts small, grows as you type..."
      autoGrow
    />
  </div>
</PreviewContainer>
