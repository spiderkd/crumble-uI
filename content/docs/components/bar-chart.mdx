## Installation

```bash
npx shadcn add https://crumble.dev/r/bar-chart.json
```

## Custom colors and format

Per-bar colors via the `color` field. `formatValue` must come from a client component — it cannot be an inline arrow function in MDX.

<PreviewContainer
  componentName="bar-chart"
  code={`<BarChart
  data={[
    { label: "Q1", value: 120, color: "oklch(0.6 0.18 260)" },
    { label: "Q2", value: 180, color: "oklch(0.6 0.16 145)" },
    { label: "Q3", value: 150, color: "oklch(0.65 0.18 55)" },
    { label: "Q4", value: 210, color: "oklch(0.62 0.22 25)" },
  ]}
  formatValue={(v) => \`$\${v}k\`}
  axisLabel="Quarterly revenue"
/>`}
>
  <BarChartRevenueDemo />
</PreviewContainer>
