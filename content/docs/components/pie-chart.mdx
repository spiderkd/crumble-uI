## Installation

```bash
npm install d3-shape d3-scale
npx shadcn add https://crumble.dev/r/pie-chart.json
```

## Custom value format

`formatValue` must come from a client component — it cannot be an inline arrow function in MDX.

<PreviewContainer
  componentName="pie-chart"
  code={`<PieChart
  formatValue={(v) => \`\${v}k\`}
  data={[
    { label: "Direct",   value: 120 },
    { label: "Organic",  value: 85 },
    { label: "Referral", value: 45 },
  ]}
/>`}
>
  <PieChartSourcesDemo />
</PreviewContainer>
